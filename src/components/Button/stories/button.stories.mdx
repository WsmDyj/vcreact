提供启动即时操作的方法，响应用户点击行为，触发相应的业务逻辑。

### 按钮样式
使用 type 属性来定义 Button 的样式。
<Preview>
  <Button>默认按钮</Button>
  <Button type="primary">主要按钮</Button>
  <Button type="danger">危险按钮</Button>
  
  <Button type="dashed">虚线按钮</Button>
  <Button type="text">文字按钮</Button>
  <Button type="link" href='//www.baidu.com' target='_blank'>链接按钮</Button>
</Preview>

### 按钮类型
* 使用 **shape** 属性设置按钮形状
* 使用 **ghost** 属性使按钮背景透明,常用在有色背景上
* 使用 **disabled** 属性来让按钮处于不可用状态

<Preview>
  <Button type="primary" shape='round'>圆角按钮</Button>
  <Button type="primary" icon={<Icon icon="bell"/>} shape='circle'></Button>
  <Button ghost type="primary">幽灵按钮</Button>
  <Button disabled>禁用按钮</Button>
</Preview>

### 按钮配合图标使用
有时为了增强按钮的提示性辅助解释功能的含义，或者加强操作的重要程度，按钮会与图标一起使用。
<Preview>
  <Button type="primary" icon={<Icon icon="spinner" pulse />}>
    加载按钮
  </Button>
  <Button type="primary" icon={<Icon icon="download" />}></Button>
  <Button type="dashed" icon={<Icon icon="download" />}></Button>
</Preview>

### 不同大小
 使用 size 属性来定义 Button 的按钮大小。
<Preview>
  <Button size="large">Large</Button>
  <Button>Default</Button>
  <Button size="small">Small</Button>
</Preview>
